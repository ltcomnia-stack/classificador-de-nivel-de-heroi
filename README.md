🦸‍♂️ Classificador de Nível de Herói

Este projeto é uma aplicação web simples desenvolvida como parte de um desafio de lógica de programação. O objetivo é classificar um herói em diferentes níveis (Ferro, Bronze, Prata, etc.) com base na quantidade de experiência (XP) inserida.

🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web:
HTML5: Estruturação da interface.
CSS3: Estilização e layout responsivo.
JavaScript: Lógica de programação e manipulação do DOM.

📋 Funcionalidades
Entrada do nome do herói.
Entrada da quantidade de experiência (XP).
Classificação automática baseada nas seguintes regras:
Ferro: XP menor que 1.000
Bronze: XP entre 1.001 e 2.000
Prata: XP entre 2.001 e 5.000
Ouro: XP entre 5.001 e 7.000
Platina: XP entre 7.001 e 8.000
Ascendente: XP entre 8.001 e 9.000
Imortal: XP entre 9.001 e 10.000
Radiante: XP maior ou igual a 10.001

💻 Como rodar o projeto
Faça o download ou clone este repositório.
Certifique-se de que os arquivos index.html, style.css e script.js estejam na mesma pasta.
Abra o arquivo index.html em qualquer navegador web.

📝 Exemplo de Saída
Ao clicar no botão "Classificar", o sistema exibirá uma mensagem como:
"O Herói de nome Arthur está no nível de Ouro"
